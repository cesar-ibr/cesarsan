Search Pattern

Given string1 and string2 look for a match of string2 in string1. Return true if there is a match otherwise return false.

Pseudo Code

        if s2 length bigger than s1 return false
        pointer = 0
        for char in s1
          if char equals s2[pointer]
            if pointer reached the end of s2 return true
            else pointer = pointer + 1
          else pointer = 0
        endfor
        return pointer == s2.length

Javascript Code

        function searchPattern (s1 = '', s2 = '') {
          if (s2.length > s1.length || !s2) {
            return false;
          }
          let pointer = 0;
          for (c of s1) {
            if (c === s2.charAt(pointer)) {
              if ((pointer + 1) === s2.length) {
                return true;
              } else {
                pointer++;
              }
            } else {
              pointer = 0;
            }
          }
          return pointer === s2.length;
        }
